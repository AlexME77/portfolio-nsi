# Projet robot GPS

Ce projet a été réalisé dans le cadre de la spécialité **Numérique et Sciences Informatiques (NSI)** en Terminale avec deux de mes camarades.

L'objectif de ce projet est de programmer un robot et le contrôler à distance via une interface web en utilisant des technologies de communication entre le robot et le navigateur. 

Ce projet m'a permis de mettre en pratique des notions d'algorithmique, de programmation et de communication réseau.

Ce projet est développé dans un dépôt séparé.
