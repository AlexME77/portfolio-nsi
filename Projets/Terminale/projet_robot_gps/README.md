# Projet robot GPS

Projet réalisé dans le cadre de la spécialité NSI (Numérique et Sciences Informatiques) en Terminale.

L'objectif de ce projet est de programmer un robot et de le contrôler à l'aide d'une page web.

Ce projet est développé dans un dépôt séparé.
